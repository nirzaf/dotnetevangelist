# Introduction about blog author

Mohamed Farook Mohamed Fazrin is a senior full-stack software engineer with over a decade of experience in various technologies. 
He is currently working as a Software Development Specialist at the Primary Health Care Corporation in Qatar, 
where he is working on a mobile application backend service, migrating REST API to GraphQL, optimizing query performance using 
Azure Redis distributed caching, and migrating data from SQL server to FHIR resources type database. 

He has previous experience as a Senior Full-stack Engineer at Quadrate Tech Solutions, where he worked on a hotel ERP solution 
and contributed to user interface development using Angular, and a Dot NET Engineer at Voigue Private Limited,
where he worked on a cloud-based phone system using .NET Core and WPF. He lists his top skills as deep listening,
mentoring and managing teams, completing projects under tight schedules and budgets, versatility, speed, and interest in learning and 
mastering innovative technologies, and staying curious, focused and craving new technologies.
